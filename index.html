CSS

  • There are three ways to style a website: Inline CSS, Internal CSS Style Sheet and External Style Sheet.
  • Inline CSS styles an element directly in the HTML file.
  • Internal Style sheet embeds CSS rules in the head
  • External Stlye sheet is the most common method. The CSS is in a seperate file which allows for easier design changes. We use the link optin to connect the file.
  • Linked files can be found in the head of a html file.

JavaScript

  • JavaScript is confusing but when it makes sense it is exciting!
  • Executes top to bottom - it is zero indexed and sequentioal
  • JavaScript is case sensitive
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • functions are a set of instructions to preform a certain task. if you see a variable name followed by () it is most likely a function.
  • functions have two steps:
    1. Define a function. This does not automatically execute the function
    2. Call the function. This allows the function to be executed.
  • the consolue is a developer tool found when inspecting a web page in browser. It is the best environment to check out code.
  • if is used as a way to introduce decision making into code.
  • for loops are predictable patterns of indicies to perform a taks on all items in an array by allowing a single code block to be executed over and over.
  • There are three steps to a for loop:
    1. starting point for loop
    2. condition
    3. execution which is found in {}
